# MJU AI & Applications Lab Website

Apple-inspired static lab website prototype.

## Files
- `index.html`: single-page shell
- `styles.css`: Apple-like photography-first visual system
- `script.js`: hash routing, project filters, publication filters, gallery detail pages, search dialog
- `assets/`: local SVG placeholder imagery

## Pages
- `#/home`
- `#/research`
- `#/projects`
- `#/projects/:area`
- `#/project/:slug`
- `#/people`
- `#/people/:slug`
- `#/publications`
- `#/publication/:slug`
- `#/gallery`
- `#/gallery/:slug`
- `#/contact`
- `#/join`

## Replacing images
Replace SVG files in `assets/` with real lab photos using the same filenames, or update paths in `script.js`.
